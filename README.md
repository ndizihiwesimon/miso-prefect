# miso-prefect

A lightweight project scaffold for working with Prefect flows.

## Overview

This repository contains code and configuration for building and running workflow orchestration with Prefect.

## Getting Started

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies.
4. Run your Prefect flow.

## Development

- Keep flows modular and reusable.
- Store configuration in environment variables where possible.
- Add tests for flow logic and tasks.
